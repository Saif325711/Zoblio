# Zoblio - Corporate Job Portal

A modern corporate job portal built with React.js, featuring a bold dark theme with golden yellow accents.

## Tech Stack
- React 18
- React Router v6
- Vite
- Vanilla CSS

## Getting Started

```bash
npm install
npm run dev
```

## Features
- Bold dark enterprise theme
- Responsive design
- Industry-focused job listings
- Role-based dashboards (Job Seeker, Employer, Admin)
